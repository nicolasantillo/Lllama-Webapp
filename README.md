# Llama-Webapp
Webapp written in Python that manages the formatting of input and output messages, using the libraries:
<ul>
  <li>llama_cpp, which uses CPU-based LLaMA models</li>
  <li>gradio, used as an interface for chatbots</li>
</ul>
